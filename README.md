# Antman

lossless (un)compress files

## Installation

```bash
git clone https://github.com/Saverio976/Antman.git
cd Antman
make
```

## Usage

### Compress

```bash
./antman/antman ./path/to/file 1 > ./file_compressed
```

### Uncompress

```bash
./giantman/giantman ./path/to/file 1 > ./file_uncompressed
```
